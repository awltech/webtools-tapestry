webtools-tapestry

XA Web Tools for Tapestry: an eclipse plugin for developping on ApacheTapestry

Features

  • Wizard to create a new page
  • Wizard to create a new component
  • Tapestry editor with java/tml files in 2 tabs
  • content-assist on tml components, mixins, pages, parameters, property expressions, messages...
  • javadoc on hover
  • navigation to source (ctrl-clic on component, mixin, page)
  • ...

Works on existing projects without config (no nature, no facet), if the source folders are defined correctly.
